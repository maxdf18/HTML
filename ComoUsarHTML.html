¡Bienvenidos a HTML para Niños!

¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es como la varita mágica para hacer páginas web. Con HTML, podemos darle forma y color a nuestras ideas y compartirlas con el mundo. ¡Es súper divertido!

Estructura básica de una página HTML

¡Presta atención a la receta para crear una página web con HTML! 


      <!DOCTYPE html>
      <html lang="es">
      <head>
        <meta charset="UTF-8">
        <title>Título de la Página</title>
      </head>
      <body>
        Contenido mágico de la página
      </body>
      </html>

¿Ves los ingredientes mágicos? ¡Ahora ya sabes cómo comenzar a crear tus propias páginas web!

Etiquetas HTML comunes

Hay muchas etiquetas mágicas que podemos usar en HTML para hacer cosas divertidas. ¡Aquí tienes algunas de ellas!

  • <h1> a <h6>: son como los títulos de los hechizos, ¡más grande es más importante!
  • <p>: ¡Este es el lugar donde puedes contar tus cuentos mágicos!
  • <a href="URL">: un portal mágico que te lleva a otros lugares.
  • <img src="URL" alt="Descripción">: aquí podemos invocar a las imágenes mágicas.
  • <ul> y <li>: ¡Estas etiquetas te ayudan a crear listas mágicas!
  • <table>, <tr>, <th> y <td>: ¡Incluso podemos crear tablas mágicas!
