What is HTML?

HTML (Hypertext Markup Language) is like a magic wand to create web pages. With HTML, we can give shape and color to our ideas and share them with the world. It's super fun!

Basic Structure of an HTML Page

Pay attention to the recipe for creating a web page with HTML! 


      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
      </head>
      <body>
        Magical content of the page
      </body>
      </html>

Do you see the magic ingredients? Now you know how to start creating your own web pages!

Common HTML Tags

There are many magical tags we can use in HTML to do fun things. Here are some of them!

  • <h1> to <h6>: They are like spell titles, the bigger, the more important!
  • <p>: This is the place where you can tell your magical stories!
  • <a href="URL">: A magical portal that takes you to other places.
  • <img src="URL" alt="Description">: Here we can summon magical images.
  • <ul> and <li>: These tags help you create magical lists!
  • <table>, <tr>, <th>, and <td>: We can even create magical tables!

Time to Play with HTML!

Now that you have the power of HTML, it's time to play! Create your own magical pages, add images of your favorite spells, and tell the world your most amazing stories. The magic is in your hands!

Remember, have fun while you learn and create your own path of digital magic!
